10. 주피터 노트북을 PDF로 만들기¶

주피터는 노트북을 다른 여러 포맷으로 내보낼 수 있다. HTML로 내보내려면 File > Download as 메뉴를 클릭하고 HTML을 선택한다. 실제로 파일 변환을 처리하는 프로그램은 nbconvert이다. 같은 방법으로 PDF를 얻을 수 있지만, 한글 텍스트가 나타나지 않는다.

참고

마크다운이 아닌, reST 같은 언어로 작성된 내용은 변환에서 제외된다는 점을 유념하라. 그래서 nbconvert에 의해 생성된 PDF에는 이 주의문이 없을 것이다.

10.1. 한글¶

한글 텍스트가 포함된 PDF를 얻으려면 주피터의 레이텍 템플릿 파일을 조작해야 한다. 다음과 같은 nbconvert 패키지의 템플릿 디렉토리(/Lib/site-packages/nbconvert/templates/latex)에서 base.tplx 파일을 연다.

\site-packages\nbconvert\templates\latex\base.tplx

그리고 16 행에 다음과 같이 추가한다.

16: \usepackage{kotex}
17: \usepackage[T1]{fontenc}

10.2. 변환 프로그램과 템플릿¶

base.tplx를 수정하지 않고 한글 문서를 만들기 위해 그리고 손쉬운 파일 변환과 스타일 변경을 위해 다음과 같은 것들이 고안되었다.

  • nb2pdf.exe: nbconvert를 이용하여 주피터 노트북을 레이텍으로 변환하고, xelatex을 이용하여 PDF를 만든다. 이 프로그램은 이전 판의 nb2pdf.ps1을 대체한다.
  • kari.tplx: nb2pdf.exe가 노트북 파일을 레이텍으로 변환할 때 이 템플릿이 사용된다.
  • karijupyter.sty: 생성된 레이텍 파일이 이 레이텍 스타일을 사용한다.

사용 방법은 다음과 같다.

#>nb2pdf.exe foo.ipynb

‘-e’ 옵션을 주면, 생성된 텍 파일에서 문자열들을 다른 것으로 교체하도록 strrep.exe가 호출된다.

#>nb2pdf.exe foo.ipynb -e

strrep.exe 프로그램에 대해 문서 생성을 위한 파이썬 스크립트를 보라.

10.2.1. 문서의 제목과 저자 정보¶

노트북의 메타데이터를 이용하여 문서의 제목과 저자를 추가할 수 있다. Edit > Edit Notebook Metadata 메뉴를 클릭하고, 나타난 창에서 다음과 같이 추가하라.

"latex": {
  "title": "How to Convert a Single Jupyter Notebook to PDF",
  "author": "KARI",
  "language": "english"
},

‘language’ 옵션의 설정이 생성되는 텍 파일에 전달된다. 아무 언어도 설정되지 않으면 한국어 문서로 간주된다.

10.2.2. 그림¶

karijupyter.sty는 \includegraphics 명령을 재정의하여 판면 폭보다 큰 이미지를 판면에 맞게 축소한다.
